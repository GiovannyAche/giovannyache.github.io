proyectos

Hospital

Se desarrollo una página web utilizando Python para el backend y JavaScript para el fronted de un hospital, con el objetivo de permitir un inicio de sesión dinámico tanto para empleados como para pacientes, teniendo en cuenta el cargo y las funcionalidades específicas de cada uno. Esta página web debe permitir realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) en los datos de pacientes, familiares y la historia clínica.

REPOSITORIO

Banco

Esta plataforma permite a los clientes de un banco realizar un inicio de sesión seguro y acceder a diversas funcionalidades. Utilizando el patrón de diseño Modelo-Vista-Controlador (MVC), he logrado una estructura organizada y modular que facilita el mantenimiento y la escalabilidad del sistema. fue desarrollada en python con ayuda de el framework django y su fronted fue desarrollado en javscript con vue

Registradurida backend

En esta etapa del proyecto, nos enfocamos en el desarrollo del backend utilizando Python y el framework Flask. Nuestro objetivo principal fue establecer conexiones con una base de datos MongoDB para crear usuarios en un sistema de votaciones, pero con un enfoque específico en los administradores
